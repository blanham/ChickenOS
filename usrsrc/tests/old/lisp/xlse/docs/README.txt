Description
-----------

XLSE stands for Extensible Lisp Scripting Engine. It is essentially a LISP
interpreter written entirely in C++, using no external libraries outside of
those provided by the C++ standard library. It was designed to be portable,
easily extensible and easily integrated into existing software solutions.

Visit http://www.cs.mcgill.ca/~mcheva/ for more information.
